javascript - 在onclick函數中傳遞字元串參數

  
0 0

我想傳遞一個參數( 例如 。 一個用於Onclick函數的字元串。 目前，我這樣做：

'<input type="button" onClick="gotoNode(' + result.name + ')"/>'

使用 result.name 例如等於字元串"添加"。 單擊這裡按鈕時，出現了一個錯誤，指出添加沒有被定義。 由於這個調用與一個數字參數完美地工作，我假設它與字元串中的符號""有關。 以前有誰有這個問題？

看起來你正在從字元串構建dom元素。 你只需要在 result.name 中添加一些引號：

'<input type="button" onClick="gotoNode('' + result.name + '')"/>'

你真的應該用正確的DOM方法來做，儘管。

var inputElement = document.createElement('input');
inputElement.type ="button"
inputElement.addEventListener('click', function(){
 gotoNode(result.name);
});
​document.body.appendChild(inputElement);​

請注意，如果這是一個循環或者什麼，result 會在事件觸發之前更改，你需要創建一個額外的作用域氣泡來隱藏更改的變數。

我建議不使用 HTML onclick 處理程序，使用一些更常見的東西，比如 document.getElementById 。

html :

<input type="button" id="nodeGoto"/>

javascript :

document.getElementById("nodeGoto").addEventListener("click", function() {
 gotoNode(result.name);
}, false);

我猜你正在用JavaScript本身創建一個按鈕。 因此，代碼中的錯誤，它將以這種形式呈現

<input type="button" onClick="gotoNode(add)"/>'

在當前狀態下，add 將被視為一個標識符，如變數或者函數調用。 你應該像這樣轉義值

'<input type="button" onClick="gotoNode('' + result.name + '')"/>'
